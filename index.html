Tentang Saya

Selama belajar di bidang informatika, saya aktif mengerjakan berbagai proyek berbasis HTML, CSS, dan JavaScript untuk meningkatkan kemampuan dalam front-end development. Saya juga menyukai eksplorasi desain antarmuka yang bersih, minimalis, dan ramah pengguna.
